# DHIS2 dashboard app skeleton
This repository can be used as a foundation to create DHIS2 dashboard apps.

## Install
```bash
npm install
# or
yarn install
```

## Develop
You need to log in to whatever `developmentServer` is set to in [src/index.js](./src/index.js#L6).
```bash
npm start
# or
yarn start
```

## Build
```bash
npm run build
# or
yarn build
```
